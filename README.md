# Text-to-Audio-Python
Transform your text to an audio using a simple python program

---

## Setup
1. Open the folder containing your file in an IDE
2. Open your terminal.
3. Create a Virtual Environment:
   For example;
   *Windows*
   ```bash
   py -3.14 -m venv venv
   ```
   *Linux/MacOS*
   ```bash
   python3 -m venv myenv
   ```
4. Activate Environment
   *Windows*
   ```bash
   ./venv/Scripts/activate
   ```
   *Linux/MacOS*
   ```bash
   source myenv/bin/activate
   ```
5. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```
---

## Run the program
1. Run the program on the terminal or press the run button
   Terminal:
   
   *Windows*
   ```bash
   python main.py
   ```
   *Linux/MacOS*
   ```bash
   python3 main.py
   ```
